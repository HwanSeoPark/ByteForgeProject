## 🌟 프로젝트 소개
**IT 교육 추천 서비스**는 사용자 맞춤형 IT 교육 강좌를 추천하고,  
효율적인 학습을 돕기 위한 웹 서비스입니다.  
사용자 정보와 선호도를 기반으로 강좌를 추천하고, 유튜브 강좌 요약 기능도 제공합니다.

## 🧩 주요 기능
- 🔎 사용자 맞춤형 IT 강좌 추천
